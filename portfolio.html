Chart 5: Indonesia Inflation

I retrieved Indonesia's inflation data from the Economics Observatory, which sources its API data from the Statistics Office of Indonesia. However, it was not immediately clear whether the data represented monthly or annual percentages. So, I took the following steps: (1) Check the data. After checking the data, I found that it reflects the month-on-month percentage change (MoM) so I edited the notes below the title. (2) Due to the fluctuation caused by seasonality, I chose to show only the end-of-period (December) data. Additionally, I limited the data to start from 2011 to align it with the GDP data above.


However, the data above is not up-to-date (the latest data is from 2019). Therefore, I created another csv dataset and uploaded it in GitHub, then built the chart from scratch. I also added a tooltip so we can hover over the line chart to display the month and value.

Interestingly, the current dataset reveals that Indonesia experienced deflation for five consecutive months. This is a rare occurence, even during Covid-19 pandemic. Some attribute it to declining purchasing power. But is that really the case?
